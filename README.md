# agendadecontatosimples
===== AGENDA DE CONTATOS ===== 1 - Adicionar 2 - Listar 3 - Remover 4 - Atualizar 5 - Buscar 0 - Sair Escolha: 

É uma agenda de contatos, para você organizar e buscar seus contatos salvos
No projeto há algumas escolhas:
1 - Adicionar (você adiciona os contatos)
2 - Listar (você lista todos os contatos salvos)
3 - Remover (você remove algum contato salvo)
4 - Atualizar (você atualiza a lista após adicionar ou remover)
5 - Buscar (você busca um contato já salvo)
0 - Sair (você sai da agenda de contatos)
